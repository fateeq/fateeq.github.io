Impurity Calculator

A Beginner Javascript Project

Summary

The Impurity Calculator is a beginner JavaScript application, the inspiration for which came from a problem I faced often at work when I was a chemist. The application is essentially a clean-looking, error-free alternative to the spreadsheet calculations my colleagues had to perform and that I had to review.

While I drew inspiration from problems at work, this application was created solely for my practice.

Tools used: HTML, CSS, JavaScript

You can interact with the application here and view the code on GitHub. For notes and rules on the calculations, please see GitHub.

Future Work

While my application is not perfect, I am happy with the amount of practice I received working on it. I decided to move on to more advanced sections of my course and learn better ways of writing code. I will be visiting this application, or something very similar to it in the future and challenge myself to build it using good coding practices.
